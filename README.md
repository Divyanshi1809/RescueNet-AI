# 🌍 Overview

Natural disasters often create chaos due to delayed communication, inefficient resource allocation, and lack of centralized emergency coordination.

**RescueNet-AI** is designed to solve these challenges by providing a centralized disaster management platform where citizens, NGOs, rescue teams, and administrators can collaborate in real time.

The system enables disaster reporting, emergency request management, AI-assisted risk monitoring, live dashboards, interactive maps, resource tracking, and alert generation—all from a single platform.

---

# ✨ Key Features

## 🔐 Secure Authentication

* JWT Authentication
* Spring Security
* Password Encryption
* Role-Based Access Control

---

## 🚨 Disaster Reporting

Users can report disasters by providing:

* Disaster Title
* Disaster Type
* Severity
* Description
* Location

Reports are stored securely in PostgreSQL and become instantly visible on the administrator dashboard.

---

## 🆘 Emergency Requests

Citizens can submit emergency help requests.

Examples:

* Medical Emergency
* Rescue Required
* Food Assistance
* Shelter Needed
* Water Supply

Administrators can monitor every request and coordinate rescue operations.

---

## 📢 Alert Management

The system allows administrators to generate emergency alerts.

Examples:

* Flood Warning
* Earthquake Alert
* Cyclone Warning
* Fire Emergency
* Evacuation Notice

Alerts are displayed instantly on the dashboard.

---

## 🏥 NGO Management

NGOs can

* Register
* Manage their profile
* Update available resources
* Participate in relief operations

---

## 📦 Resource Management

Track emergency resources including

* Food

* Medicines
* Blankets
* Rescue Equipment
* Medical Kits

This helps administrators understand available supplies during emergencies.

## 🗺 Interactive Disaster Map

# 🚀 Getting Started


---

## Backend

```
cd backend
```

Configure PostgreSQL in

```
application.properties
```

Run

```
mvn spring-boot:run
```

Server

```
http://localhost:8089
```

---

## Frontend

```
cd frontend
npm install
npm start
```

Frontend

```
http://localhost:3000
```
